# В Швейцарии проголосовали за запрет на строительство новых АЭС
Согласно информации, большинство жителей Швейцарии поддержали "Энергетическую стратегию-2050", согласно которой новые атомные электрастанции больше не будут строиться в стране. На референдуме в Швейцарии жители проголосовали за запрет на строительство новых АЭС в пользу возобновляемых источников энергии. Об этом свидетельствуют предварительные данные всенародного голосования. По данным на 16.00 по Киеву, 58,5% жителей Швейцарии поддержали "Энергетическую стратегию-2050", согласно которой новые атомные электростанции больше не будут строиться в стране. По этому вопросу не требовалось получение кантонального большинства. Согласно одобренной стратегии, правительство Швейцарии теперь будет также поддерживать разработку возобновляемых источников энергии для использования ее вместо атомной. Кроме этого, запланирована программа по сокращению потребления энергии за счет внедрения новых энергетических стандартов, а также налоговых привилегий для тех, кто занимается энергетическими инновациями. Граждане Швейцар уже во второй раз голосуют по вопросу национальной энергетической стратегии. В ноябре 2016 года они проголосовали за то, что действующие швейцарские АЭС должны работать дальше, до конца срока своей службы.
